# 🧮 Calculadora Interativa em C++

Projeto desenvolvido para praticar lógica de programação em C++.

## 🚀 Funcionalidades

- Soma (+)
- Subtração (-)
- Multiplicação (*)
- Divisão (/)
- Validação de divisão por zero

## 💻 Como usar

1. Execute o programa
2. Digite dois números
3. Escolha a operação desejada
4. Veja o resultado na tela
